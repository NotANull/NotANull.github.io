Desarrollo Web

Esta aplicación está hecha con HTML, CSS y JavaScript. El proyecto está estructurado por sectores en donde cada parte tiene sección, a medida que iba enmaquetando también iba aplicando estilos y, con un poco de ayuda de Bootstrap, la página web fue tomando forma y color. Como toda página tiene que tener su comportamiento, también se le aplicó JavaScript aunque no en gran medida porque se trata de una página de presentación pero cuando vaya haciendo proyectos más complejos, verán muchos más comportamientos complejos y robustos. Con el paso del tiempo cuando vaya mejorando mis habilidades como desarrollador, voy adaptando el proyecto para que se vea lo más profesional posible y si bien es básico a los ojos de gente con experiencia y expertos, para mi es un gran avance como desarrollador entusiasta.
